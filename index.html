CSCE 242: Website Applications

Nhan Do

CSCE 242 is a course offered here at UofSC which teaches students how to build a front-end of a website by using both HTML and CSS. This course will also implement the usage of JavaScript to allow more functions for the website. With the ultilzation of JavaScript, this will also teach students how to ultilze nodes as well to store and retrieve server data.

Assignments

Assignment 1

In this assignment, we learned how to create a website using basic html and sourcing images for the website.

Assignment 2

In this assignment, we used CSS to create styling for a website. We learned how to incorporate color schemes and images for a website background. This assignment also introduces the usage of a navigation bar.

Assignment 3

In this assignment, we used CSS to create flex containers to enhance Website styling. This assignment also teaches how to use Media Query so that the website can display veritcally on smaller screens.
